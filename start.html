GETTING STARTED

Download the newest version from the GitHub repository.

Github Repository

You'll need 3 files:
JsonWax.h, JsonWaxEditor.h & JsonWaxParser.h
Put all 3 files in your Qt-project folder. You'll only need to:

include "JsonWax.h"

in your project to use JsonWax.

If you haven't enabled C++11 support in your project, you'll need to include this text in your .pro file:

CONFIG += c++11

HOW TO USE IT

JsonWax is based on QVariants which can contain basic data types such as integer, string, double, bool. To retrieve a value from a QVariant, you convert it .toString(), .toInt(), etc.
A QVariant is stored using setValue(keys, value) and retrieved using value(keys).

JsonWax json;
json.setValue({"A",4,"B",1}, "test");
qDebug() << json.value({"A",4,"B",1}).toString();
json.saveAs("basil.json");

Result (contents of the file "basil.json"):

{
  "A": [
    null,
    null,
    null,
    null,
    {
      "B": [
        null,
        "test"
      ]
    }
  ]
}

QVariantList keys may contain only ints (larger than -1) and QStrings (any UTF-8 characters), as it describes a location within a JSON-document. If you use an int, it specifies an element in a QList (where 0 is the first element). If you use a QString, it specifies an element in a QMap.
You can't access the maps and lists directly, but you can use various functions to navigate your documents.
